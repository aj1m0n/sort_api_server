# HOW TO

## sort-api-server

1. apiリポジトリをworkspaceにクローンする．
2. mkdir images
3. ip install flask
4. python3 sort-api-server

## demo.py
1. demo.pyのipcamera, udp flagをTrue
2. 76行目のアドレスを任意の値(xx)にする．(192.168.xx.255
3. dockerコンテナ内で 

```
python3 demo.py
```
